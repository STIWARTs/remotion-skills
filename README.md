Welcome to your Remotion project!

## Commands

**Install Dependencies**

```console
npm i
```

**Start Preview**

```console
npm run dev
```

**Render video**

```console
npx remotion render
npx remotion render MainVideo out/MainVideo.gif --scale=0.25
```

**Upgrade Remotion**

```console
npx remotion upgrade
```
